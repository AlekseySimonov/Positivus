# Prositivus site  


## Tech Stack  
**Client:** Bootstrap, Scss

## Run Locally  
Clone the project  

~~~bash  
  git clone https://github.com/AlekseySimonov/Positivus
~~~

Go to the project directory  

~~~bash  
  cd my-project
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start the server  

~~~bash  
npm start
~~~
